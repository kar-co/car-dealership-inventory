# Project Structure

- `backend/fastapi_backend/app`: API routers, schemas, models, database, security, and dependencies.
- `backend/fastapi_backend/alembic`: migration environment and revisions.
- `backend/django_backend`: existing Django administration project.
- `docs`: project documentation.
- `frontend/src`: React authentication, dashboard, API client, and Tailwind styles.
- `frontend/vite.config.js`: development API proxy configuration.
