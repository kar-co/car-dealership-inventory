# Frontend

The Vite React application uses HTML5, CSS3, and Tailwind CSS. It contains exactly Login, Register, and Dashboard. Dashboard loads protected vehicle data, offers text and price-range search plus category filtering, disables a purchase at zero stock, and presents responsive loading/error states.

Run `npm install`, `npm run dev`, and `npm run build` from `frontend`. The default `/api` URL proxies to `http://localhost:8000`; set `VITE_API_BASE_URL` for another deployment target.
